# ⚠️ BLOG-ARTIKEL TEMPLATE – NICHT BEARBEITEN

**Dieser Ordner enthält die Master-Vorlage für alle Blog-Artikel.**

## Regeln

- **Nicht bearbeiten** – Diese Dateien dienen nur als Kopiervorlage
- **Neue Artikel** – Immer von hier kopieren, nie von `test-artikel/`
- **Layout-Änderungen** – Nur hier anpassen, dann in neue Artikel übernehmen

## Verwendung

```
blog/_template/index.html  →  Kopieren nach  →  blog/[artikel-slug]/index.html
```

Beim Kopieren: Pfade prüfen (`../` und `../../` bleiben gleich, da Struktur identisch).

## Zugehörige Dateien

- `blog/article-styles.css` – Gemeinsame Artikel-Styles (wird von allen Artikeln geladen)
- `blog/ARTIKEL-VORLAGE.md` – Anleitung für neue Artikel
